# 🧪 A/B Testing Framework

A reusable, end-to-end experimentation toolkit for marketing and product analysts.

## Structure
- `src/ab_tests.py` — All reusable functions
- `notebooks/01` — Sample size & power calculations
- `notebooks/02` — Data validation & sanity checks
- `notebooks/03` — Statistical significance testing
- `notebooks/04` — Business recommendation memos
- `data/` — Synthetic experiment datasets (3 scenarios)
- `outputs/` — Charts and stakeholder memos

## Quickstart
pip install -r requirements.txt
python data/generate_data.py
jupyter notebook notebooks/
